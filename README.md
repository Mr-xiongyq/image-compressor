# 图片压缩工具

一个简单易用的在线图片压缩工具，无需上传到服务器，所有操作均在浏览器本地完成，保护您的隐私。

## ✨ 特性

- 🔒 纯浏览器端压缩，无需上传服务器
- 📱 响应式设计，完美支持移动端
- 🌓 自动适配深色/浅色模式
- 🖼 支持 JPG、PNG 格式图片
- 👀 实时预览压缩效果
- 📊 显示压缩前后文件大小对比
- ⚡️ 快速且高效的压缩算法
- 🎨 现代化的用户界面
- 🔍 支持高分辨率屏幕显示

## 🚀 使用方法

1. 打开网页即可使用，无需安装任何软件
2. 选择图片方式：
   - 点击上传按钮选择图片
   - 直接将图片拖拽到上传区域
3. 调整压缩质量：
   - 使用滑块调节压缩质量（0-100%）
   - 实时预览压缩效果
4. 下载压缩后的图片：
   - 点击"下载压缩后的图片"按钮
   - 自动添加"compressed_"前缀以区分原图

## 💡 技术特点

- **纯前端实现**：
  - 使用 HTML5 Canvas 进行图片处理
  - 利用 File API 实现本地文件操作
  - 使用 Blob API 处理二进制数据

- **现代化设计**：
  - 采用 CSS Grid 和 Flexbox 布局
  - 使用 CSS 变量实现主题切换
  - 添加平滑的过渡动画效果

- **多设备支持**：
  - 自适应布局设计
  - 触摸屏优化
  - 支持各种主流浏览器

## 🛠 本地开发

1. 克隆项目：
   ```bash
   git clone https://github.com/Mr-xiongyq/image-compressor.git
   ```

2. 进入项目目录：
   ```bash
   cd image-compressor
   ```

3. 使用本地服务器运行（推荐使用 Live Server）：
   - VS Code: 安装 Live Server 扩展
   - 或使用 Python 简单服务器：
     ```bash
     python -m http.server 8080
     ```

## 📝 注意事项

- PNG 格式图片会自动转换为 JPEG 格式以获得更好的压缩效果
- 建议图片尺寸不超过 2048x2048 像素
- 过大的图片会自动等比例缩小
- 为保证压缩质量，单个文件大小建议不超过 10MB

## 🔒 隐私说明

- 所有操作均在浏览器本地进行
- 不会上传图片到任何服务器
- 不会保存任何用户数据
- 不使用 Cookies 或本地存储
