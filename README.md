# 图片压缩网站

一个简单易用的在线图片压缩工具。

## 主要功能

1. 支持上传 PNG、JPG 等格式的图片文件
2. 根据用户需求对图片进行压缩，减小文件大小
3. 实时预览压缩前后的图片效果
4. 显示压缩前后的文件大小对比
5. 支持压缩后图片的下载

## 技术栈

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript
- 响应式设计

## 使用说明

1. 打开网站
2. 点击上传按钮或将图片拖拽到指定区域
3. 选择压缩比例
4. 预览压缩效果
5. 下载压缩后的图片

## 开发规范

- 使用语义化 HTML 标签
- 代码符合 W3C 标准规范
- 所有代码添加中文注释
- 采用响应式设计，适配不同设备
